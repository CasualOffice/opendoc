## Supported Foundation

Phase 0 provides:

- deterministic normalized paragraph/text values and stable IDs;
- strict bounded normalized JSON v0 load/export;
- grapheme-aware insert/delete and paragraph split/join;
- semantic inverses, undo/redo, revision checks, and position maps;
- canonical directed selection mapped through edits and history;
- bounded ordered transaction and selection event delivery;
- bounded DOCX ZIP admission, deterministic part metadata, and verified
  on-demand part reads;
- reproducible package/model performance reports;
- generated package/security corpus and fuzz infrastructure.

## Deferred Capabilities

These are intentionally not Phase 0 claims:

- WordprocessingML semantic import, styles, themes, numbering, sections, media
  references, deterministic semantic snapshots, and compatibility reports:
  Phase 1A;
- shaping and paragraph layout: Phase 1B;
- pagination and display lists: Phase 1C;
- rendering, visual baselines, and hit testing: Phase 1D;
- DOCX writing, edit/save/reopen workflows, and semantic round-trip baselines:
  Phase 2;
- IME, pointer editing, clipboard, advanced office features, plugins, and
  collaboration: later roadmap phases;
- public stable API or production integration: later release gates.
